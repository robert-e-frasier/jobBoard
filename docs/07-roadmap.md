# 07 - Roadmap

## Phase 1 - Project Foundation

* Create repository structure.
* Configure React and TypeScript frontend.
* Configure Node.js and Express backend.
* Configure PostgreSQL database.
* Establish project architecture and documentation.

## Phase 2 - Authentication

* Email/password registration.
* Email/password login.
* Continue with Google authentication.
* Session management.
* Role selection during registration.

## Phase 3 - User Profiles

### Job Seeker

* Create profile.
* Edit profile.
* Upload resume.
* Manage personal information.

### Recruiter

* Create recruiter profile.
* Create company profile.
* Edit company information.

## Phase 4 - Job Management

* Recruiters create job postings.
* Recruiters edit job postings.
* Recruiters close or remove job postings.
* Public users browse available jobs.

## Phase 5 - Applications

* Job seekers apply to jobs.
* Recruiters review applications.
* Recruiters update application status.
* Job seekers track submitted applications.

## Phase 6 - Search and Filtering

* Keyword search.
* Location filtering.
* Employment type filtering.
* Remote, hybrid, and on-site filtering.
* Salary range filtering.

## Phase 7 - Notifications

* Email notifications.
* Application status updates.
* Recruiter notifications.
* Job seeker notifications.

## Phase 8 - Polish

* Responsive design improvements.
* Accessibility improvements.
* Performance optimization.
* Error handling.
* Loading states.
* User feedback improvements.

## Phase 9 - Deployment

* Deploy frontend.
* Deploy backend.
* Deploy PostgreSQL database.
* Configure production environment variables.
* Configure monitoring and logging.

## Long-Term Goals

The public repository is intended to demonstrate professional software engineering practices and serve as a portfolio project.

Additional proprietary functionality may be developed separately in a private repository while preserving this project's ability to function as a complete standalone application.
